## PHP Data types
-   Integer
-   Float
-   String
-   Boolean
-   Type casting is also an option
-   <a href="{{ site.baseurl }}notes/PHP-Arrays">PHP Arrays</a>
```php
<?php
$a = (int) $b; //changing to a integer  
$a = (string) $b; //changing to a string  
$a = (float) $b; //changing to a real number
?>
```
